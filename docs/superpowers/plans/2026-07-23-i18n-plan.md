# obsidian-installer-plugin i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add English / Traditional Chinese (Taiwan) support to `obsidian-installer-plugin`'s UI, auto-detected from Obsidian's own language setting.

**Architecture:** A new `src/i18n/` module holds two flat string dictionaries (`en.ts`, `zh-TW.ts`) plus `detectLocale`/`setLocale`/`t` helpers. `main.ts#onload()` detects the locale once at startup and calls `setLocale()` before anything renders. Every literal UI string in `settingsTab.ts` and `main.ts` is replaced with a `t('key', vars?)` call.

**Tech Stack:** TypeScript, Vitest (existing `node` test environment — no DOM needed for the i18n module itself).

## Global Constraints

- Spec: `docs/superpowers/specs/2026-07-23-i18n-design.md` — read it before starting.
- Out of scope: `manifest.json` description, `docs/index.html`, live language switching without reload.
- `en.ts` and `zh-TW.ts` must have identical key sets at all times — enforced by a parity test.
- Do NOT `git commit` or `git push` anything in this plan — this project's standing rule requires an explicit user request before any commit. Stop after the final verification task and report results instead.

    expect(detectLocale('zh-CN')).toBe('zh-TW');

    expect(detectLocale('en')).toBe('en');
    expect(detectLocale('ja')).toBe('en');
    expect(detectLocale(null)).toBe('en');
  });

describe('t', () => {
  beforeEach(() => {
    setLocale('en');
  });

  it('returns the plain string for a key with no vars', () => {
    expect(t('registry.heading')).toBe('Available in mirror');
  });

  it('interpolates vars', () => {
    expect(t('notice.removed', { id: 'my-plugin' })).toBe('Removed my-plugin');

    setLocale('zh-TW');

- [ ] **Step 5: Write `src/i18n/index.ts`**

```ts
import { en } from './locales/en';
import { zhTW } from './locales/zh-TW';

export type Locale = 'en' | 'zh-TW';
export type TranslationKey = keyof typeof en;

const dictionaries: Record<Locale, Record<string, string>> = {
  en,
  'zh-TW': zhTW,
};

let currentLocale: Locale = 'en';

export function detectLocale(raw: string | null): Locale {
  if (raw === 'zh' || raw === 'zh-TW' || raw === 'zh-CN' || raw === 'zh-HK') {
    return 'zh-TW';
  }
  return 'en';
}

export function setLocale(locale: Locale): void {
  currentLocale = locale;
}

export function t(key: TranslationKey, vars?: Record<string, string | number>): string {
  const template = dictionaries[currentLocale][key] ?? dictionaries.en[key] ?? key;
  if (!vars) return template;
  return template.replace(/\{(\w+)\}/g, (match, name: string) => {
    const value = vars[name];
    return value === undefined ? match : String(value);
  });
}
```
